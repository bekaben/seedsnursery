# 

SeedNUrsery  a project stated within the WizNet Surf5 contest as a proof of concept for plant nursery facilities. We combine two module WizPoE-P1 for power and Surf5 to deal with sensor and the network.

## Used toolchain

- MDK-ARM (KEIL 5)
- Keil uVision 5
